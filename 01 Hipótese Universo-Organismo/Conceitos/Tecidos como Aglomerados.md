# Tecidos como Aglomerados

Parent:: [[MOC - Conceitos]]
Friends:: [[Órgãos como Galáxias]], [[Células como Sistemas Estelares]], [[Galáxias como Órgãos]]

## Ideia

Tecidos são conjuntos de células com padrões de vizinhança, função e comunicação. Na hipótese, eles podem ser comparados a aglomerados: regiões onde muitas unidades próximas produzem uma identidade coletiva.

Um tecido não é uma célula gigante. É uma comunidade organizada. Um aglomerado também não é uma galáxia gigante; é um padrão de agrupamento que muda a dinâmica de cada membro.

## Função

O tecido cria contexto. Ele diz a cada célula que tipo de comportamento faz sentido: contrair, secretar, proteger, transmitir, reparar, absorver.

Do mesmo modo, um aglomerado cósmico altera o destino das galáxias que o compõem. Proximidade muda trajetória, troca, pressão e forma.

## Ponte

Esta nota conecta [[Células como Sistemas Estelares]] a [[Órgãos como Galáxias]]. Ela mostra que entre unidade e órgão existe uma camada coletiva, intermediária, fundamental.

## Imagem

Tecido é constelação viva. Aglomerado é tecido gravitacional.
