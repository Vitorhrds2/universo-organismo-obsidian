Parent:: [[Atlas - Universo como Organismo]]

## Plugins instalados no vault

Instalei estes plugins em `.obsidian/plugins/` e deixei ativados em `.obsidian/community-plugins.json`:

| Plugin | Uso neste projeto |
|---|---|
| Dataview | consultas vivas por tags, tipo, escala e estado |
| Excalidraw | desenhos livres, mapas e diagramas editáveis |
| ExcaliBrain | navegação da rede como mapa mental estruturado |

## Como ativar no Obsidian

1. Reinicie o Obsidian ou use `Reload app without saving`.
2. Abra Settings.
3. Vá em Community plugins.
4. Se aparecer Restricted Mode, permita Community plugins.
5. Confirme se Dataview, Excalidraw e ExcaliBrain estão ligados.

## Fluxo recomendado

- Use [[Painel Dataview - Hipótese Universo-Organismo]] para ver a base como banco de dados.
- Use [[Mapa Visual - Corpo do Universo.canvas]] para explorar a hipótese visualmente.
- Use [[Galeria de Mídias - Universo-Organismo]] para abrir o GIF e o trailer com embeds nativos do Obsidian.
- Use [[Exploração Visual - Hipótese.canvas]] se o Obsidian abrir o canvas ativo que estava vazio.
- Use a paleta de comandos e procure `ExcaliBrain` para abrir uma navegação dinâmica.

## Depois de ativar os plugins

Os plugins passam a ler as notas depois de ativados, mas não recriam canvases vazios automaticamente. Por isso o canvas ativo foi preenchido manualmente com nós e conexões reais, e o mapa principal foi expandido com grupos, labels e cartões de arquivo.

## Nota de segurança

Plugins comunitários rodam código de terceiros dentro do Obsidian. Instalei apenas plugins populares e diretamente dos releases públicos dos autores, mas vale manter o hábito de revisar plugins antes de adicionar novos.

## Ideia futura

Se você quiser uma camada de inteligência semântica com embeddings, o próximo candidato seria um plugin de conexões semânticas. Eu deixaria isso como etapa separada, porque normalmente exige chave de API, modelo local ou permissões adicionais.
